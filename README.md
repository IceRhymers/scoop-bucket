# IceRhymers Scoop Bucket

Scoop bucket for Databricks AI Gateway proxy CLIs.

## Install

```powershell
scoop bucket add icerhymers https://github.com/IceRhymers/scoop-bucket
scoop install databricks-claude
scoop install databricks-codex
scoop install databricks-opencode
```

## Packages

| Package | Description |
|---|---|
| `databricks-claude` | Databricks AI Gateway proxy for Claude Code |
| `databricks-codex` | Databricks AI Gateway proxy for Codex CLI |
| `databricks-opencode` | Databricks AI Gateway proxy for OpenCode |

## Update

Manifests are updated automatically when new releases are published. To update installed packages:

```powershell
scoop update databricks-claude
scoop update databricks-codex
scoop update databricks-opencode
```
